# **ğŸ§  Key Insights** 
- Return rates are disproportionately driven by **new customers**  
- **Size issues account for ~45% of all returns**, making them largely preventable  
- **Instagram Shop + New Customers + COD** forms the highest-risk combination  
- Promotional spikes increase order volume but degrade order quality  
- Returning customers demonstrate significantly lower return behavior  

---

# **âœ… Business Recommendations** 
- Improve size guidance and fit transparency for high-return categories  
- Restrict COD for high-risk customer and channel combinations  
- Introduce discount guardrails during acquisition-heavy campaigns  
- Focus return reduction efforts on specific segments instead of blanket policies
